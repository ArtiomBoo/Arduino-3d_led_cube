## Parts
- 125x PL9823 (for 5x5x5 cube) 
- 1x Arduino Nano
- 1x Toggle Switch
- 1x DC Jack
- 1x 10k Potentiometer
- 5V Power Supply
- Silvered Copper wire
- materials to make a box
